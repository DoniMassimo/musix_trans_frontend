# Song Translator – Frontend

🚧 **Status:** Work in Progress

Frontend of the **Song Translator** web app, an application that allows users to view automatically generated translations of song lyrics.

The app communicates with a backend service responsible for retrieving lyrics, translating them, and storing them in the database.

---

## Demo

![Frontend demo](assets/demo/demo.gif)

---

## Tech Stack

- **Flutter** – UI framework
- **Hive** – local database
- **MVVM** – application architecture pattern

---

## Features

✔ View saved translations  
✔ Local caching of translations  
🚧 Create translations from a Spotify link  
🚧 Interaction with translations
