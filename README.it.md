# Song Translator – Frontend

🚧 **Status:** Work in Progress

Frontend della web app **Song Translator**, un'applicazione che permette di visualizzare le traduzioni automatiche dei testi delle canzoni.

L'app comunica con il backend che si occupa di recuperare i lyrics, tradurli e salvarli nel database.

---

## Demo

![Frontend demo](assets/demo/demo.gif)

---

## Tech Stack

- **Flutter** – UI framework
- **Hive** – database locale
- **MVVM** – architettura dell'applicazione

---

## Features

✔ Visualizzazione delle traduzioni salvate
✔ Cache locale delle traduzioni
🚧 Creazione traduzioni da link Spotify
🚧 Interazione con le traduzioni
